India Air Quality & Health Impact Analytics (2015-2025)

📌 Project Overview

- This project provides a comprehensive 11-year longitudinal analysis of air quality trends across India and their direct correlation with public health outcomes. Using Power BI and Python, I visualized how AQI levels, industrialization, and seasonal changes impact respiratory health across different states and zones.

⚠️ The Data Challenge (Why this project?)

- While official government portals are the primary source for environmental data, they often contain significant data gaps and missing records for specific years or smaller regions.

- To ensure the integrity of this 11-year study:

- I utilized a consolidated dataset that harmonizes various sources to provide a continuous monthly record.

- I performed extensive Data Cleaning in Python to handle inconsistencies that are common in raw environmental sensor data.

🛠️ Tech Stack

- Data Processing: Python (Pandas)

- Visualization: Power BI Desktop

- Analysis Metrics: AQI, PM2.5, PM10, Respiratory Admissions (per 100k), and Premature Deaths.

📊 Key Dashboard Features

- Interactive Geographic Map: Identifies pollution hotspots across Indian states.

- Industrialization "Staircase": A column chart proving the direct link between industrial density and AQI levels.

- Pollutant Duo Analysis: A state-wise breakdown comparing PM2.5 (fine particulate/smoke) vs PM10 (coarse particulate/dust).

- Health Burden Treemap: Visualizing how "Winter" in the "North Zone" accounts for the highest health risks.

💡 Key Insights

- Industrial Impact: Average AQI increases by nearly 40% when moving from "Low" to "High" industrial zones.

- Seasonal Danger: Respiratory admissions show a statistically significant spike during the winter months, correlating with "Severe" AQI levels.

- Component Breakdown: Many northern states show a higher ratio of PM2.5, suggesting industrial/combustion sources rather than just dust.
